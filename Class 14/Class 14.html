Reactome

Lots of folks like the reactome web interface. You can also run this as an R function but lts look at the website first. < https://reactome.org/ >

The website wants a text file with one gene symbol per line of the genes you want to map to pathways.

sig_genes <- res[res$padj <= 0.05 & !is.na(res$padj), ]$symbol
